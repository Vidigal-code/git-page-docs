# Primeiros passos

Este guia ajuda voce a rodar e personalizar o projeto rapidamente.

## Requisitos

- Node.js 20+
- npm ou pnpm

## Primeiro uso

1. Instale dependencias: `npm install`
2. Gere a base do projeto: `npm run gitpagedocs`
3. Inicie desenvolvimento: `npm run dev`

## Estrutura do projeto

- `gitpagedocs/config.json` configuracao principal
- `gitpagedocs/docs/{lang}` arquivos markdown multi-idioma
- `public/layouts` temas e templates compartilhados
