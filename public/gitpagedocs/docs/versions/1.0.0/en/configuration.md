# Configuration

All runtime options are configured in `gitpagedocs/config.json`.

## Site settings

- `defaultLanguage`: default language code
- `HideThemeSelector`: hides theme selector when `true`
- `ThemeDefault`: default selected theme id
- `ActiveNavigation`: enables `Ctrl+K` navigation
- `IconImageMenuHeader`: logo/icon path for header

## Routing

Use `routes` to map markdown files for each language.
Use `menus-header` to map titles and click paths for each language.
