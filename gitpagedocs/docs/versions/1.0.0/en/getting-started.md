# Getting Started

This guide helps you run and customize the project quickly.

## Requirements

- Node.js 20+
- npm or pnpm

## First run

1. Install dependencies: `npm install`
2. Generate project base: `npm run gitpagedocs`
3. Start development: `npm run dev`

## Project structure

- `gitpagedocs/config.json` main project configuration
- `gitpagedocs/docs/{lang}` multilingual markdown files
- `public/layouts` shared themes and templates
