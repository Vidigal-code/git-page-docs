# Configuracao

Todas as opcoes de execucao ficam em `gitpagedocs/config.json`.

## Configuracoes do site

- `defaultLanguage`: idioma padrao
- `HideThemeSelector`: oculta seletor de tema quando `true`
- `ThemeDefault`: id do tema inicial
- `ActiveNavigation`: habilita navegacao com `Ctrl+K`
- `IconImageMenuHeader`: caminho do icone/logo no cabecalho

## Rotas

Use `routes` para mapear arquivos markdown por idioma.
Use `menus-header` para mapear titulo e caminho de clique por idioma.
