# Configuracion

Todas las opciones de ejecucion estan en `gitpagedocs/config.json`.

## Configuracion del sitio

- `defaultLanguage`: idioma por defecto
- `HideThemeSelector`: oculta selector de tema cuando es `true`
- `ThemeDefault`: id del tema inicial
- `ActiveNavigation`: habilita navegacion con `Ctrl+K`
- `IconImageMenuHeader`: ruta del icono/logo en el encabezado

## Rutas

Usa `routes` para mapear archivos markdown por idioma.
Usa `menus-header` para mapear titulo y ruta de clic por idioma.
