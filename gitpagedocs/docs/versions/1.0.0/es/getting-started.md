# Primeros pasos

Esta guia te ayuda a ejecutar y personalizar el proyecto rapidamente.

## Requisitos

- Node.js 20+
- npm o pnpm

## Primer uso

1. Instala dependencias: `npm install`
2. Genera la base del proyecto: `npm run gitpagedocs`
3. Inicia desarrollo: `npm run dev`

## Estructura del proyecto

- `gitpagedocs/config.json` configuracion principal
- `gitpagedocs/docs/{lang}` archivos markdown multilenguaje
- `public/layouts` temas y plantillas compartidas
